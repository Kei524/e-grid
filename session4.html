SESSION 4 ／ 90分

アウトプット・PM × AI ＋ 総合演習

資料作成・プレゼン・PMのエッセンスをAI活用とあわせて学び、総合演習で一連の流れを体験する。

この回のゴール

  • 資料作成・プレゼン・PMの「型」のエッセンスを理解する
  • AIで構成・推敲・議事録・タスク分解を効率化できる
  • 総合演習で、課題発見から提案・発表までを通しで体験する

タイムテーブル

  • 0–10分　前回課題の共有・ふりかえり
  • 10–35分　講義：資料作成・プレゼン・PM基礎のエッセンス
  • 35–45分　講義：アウトプットづくりでのAI活用
  • 45–80分　総合演習（チーム）＋発表
  • 80–90分　全体ふりかえり・最終課題の説明

1. ドキュメンテーション（資料作成）

  • 目的から逆算：「誰に・何を・どうしてほしい」を決めてから作る
  • いきなり清書しない：構成（ストーリー）→ ラフ → 清書の順
  • 1スライド＝1メッセージ：見出しは「テーマ」でなく「言い切りの主張」
  • 図解する：比較は棒グラフ、推移は折れ線、関係は図で
  • ファクトと解釈を分ける：数字（事実）→ だから何（So What）をセットで、出典明記

2. プレゼンテーション

  • ゴールは「合意・行動」：うまく話すことより、相手が動くこと
  • 結論・全体像から：最初に要点とアジェンダで「地図」を渡す
  • Q&A：結論から答える／分からないことは持ち帰る／想定問答を準備
  • リハーサル：声に出して時間を計る

3. プロジェクトマネジメント基礎

  • QCD：品質・コスト・納期はトレードオフ。優先順位を関係者で合意
  • スコープ：「やる／やらない」を最初に握る
  • WBS：大きな仕事を作業まで分解し、担当・期限をつける
  • 報連相：悪い情報ほど早く。抱え込まずエスカレーション
  • ふりかえり（KPT）：Keep / Problem / Try で学びを残す

4. アウトプットづくりでのAI活用

場面AIの使い方
資料構成ストーリーの叩き台・各ページのキーメッセージ案を出す
推敲冗長な文の短縮・表記ゆれの統一・分かりやすい言い換え
議事録メモから要点・決定事項・ToDoを整理
PMタスクの洗い出し（WBS叩き台）・想定リスクの列挙
プレゼン想定質問（Q&A）の洗い出し

🤖 プロンプト例（資料構成のたたき台）

「アプリ会員の継続率改善」をテーマにした提案資料の構成案を作ってください。 聞き手はクライアントの事業責任者。結論ファーストで、 各スライドの「キーメッセージ（言い切りの一文）」を箇条書きで示してください。 スライドは7枚以内に。

⚠️ 注意

AIの構成・文章は「整っているが中身が一般論」になりがち。キーメッセージとファクトは自分の分析で埋める。AIは型・体裁を整える役、中身の責任は自分。

総合演習（チーム）

これまで学んだ型とAI活用を総動員して、課題発見から提案・発表までを体験します。

お題例：第3回のミニケース（飲食店アプリの会員継続率）について、事業責任者に向けた提案（3〜5枚）をチームで作成し、発表する。

✏️ 総合演習：提案づくり → 発表（35分）

  1. 問題解決：問題・原因・課題・打ち手を整理（第3回の型）
  2. 資料化：1スライド1メッセージで提案にまとめる（AIで構成・推敲）
  3. 発表：結論ファーストで各チーム3分プレゼン ＋ 質疑1問
  4. AIをどこで使い、どこを自分たちで判断したかも共有する

最終課題（研修のまとめ）

📝 最終課題：ミニ提案資料を仕上げる

総合演習で作った提案（または各自のテーマ）を、個人で提案資料（3〜5枚）に仕上げて提出。次を満たすこと。

  • 問題解決の流れ（問題 → 原因 → 課題 → 打ち手）が追える
  • 各ページがキーメッセージ（言い切り）になっている
  • ファクトと解釈が分かれ、出典が明記されている
  • AI活用レポート：使ったAI／使った場面／検証・修正した点を1段落で添える

研修全体のふりかえり（KPT）

4回を通じて学んだことを、KPTで整理して締めくくります。

  • Keep：身についた型・続けたいこと
  • Problem：まだ苦手なこと
  • Try：明日の業務で試す型・AI活用を1つ決める

チェックリスト

  • ☐ 資料を「誰に・何を・どうしてほしい」から構成できる
  • ☐ 1スライド1メッセージ・キーメッセージを言い切りで書ける
  • ☐ 結論ファーストでプレゼンし、質疑に答えられる
  • ☐ WBS・QCD・報連相などPMの基礎を理解した
  • ☐ AIで構成・推敲・WBSを効率化しつつ、中身は自分で責任を持てる
