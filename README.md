# コンサル基礎研修 ロジカルシンキング編（イーグリッド）

イーグリッド AIコンサルティング組織の新人向け連続研修「**ロジカルシンキング編**」のWeb教材です。
**90分 × 4回**（毎週開催＋間に実務適用週）、各回に**演習**と**宿題**があり、全回で**生成AI（Claude／Deep Research）併用**を学びます。

対象：既存エンジニアのリスキル ＋ 中途未経験者（前提知識は不要）。
コンセプト：**経営層と対話するための「構造」を、毎週90分で身につける**。

カリキュラムは **Web上で閲覧できる静的サイト（HTML）** として作成しており、ブラウザで開くだけで閲覧できます。GitHub Pages で公開済みです。

### 共通資料（全ページのナビから参照可能）

- トップ [index.html](index.html)：全体構成・コードレビューの比喩・修了判定
- [AI併用ガイド](ai-guide.html)：安全な使い方・プロンプトの型・AIの役割
- [フレーム選択マップ](toolmap.html)：「状況 → 使う道具」の対応表（道具の使い分け）
- [採点ルーブリック](rubric.html)：ピラミッド／ツリーの自己・相互採点表（印刷して配布可）

### 研修設計の工夫（最適化で追加）

- **第0回（事前学習）** で“崖”を緩和し、未経験者でも第1回に入りやすくした
- 全回に **「まず自力 → そのあとAI」の鉄則バナー**を固定し、演習タスクを 自力／AI で明示
- 発表できない大多数のために、各回末に **採点ルーブリックでの自己・相互採点**を組み込み
- 演習素材が「地方製造×DX」に偏らないよう、第2回に **毛色の違うBtoC成長ケース** を1つ追加
- 道具の使い分けに迷ったとき用に **フレーム選択マップ**を用意

## 特徴（わかりやすさ・図表）

- 各回の **90分タイムテーブルをバー図**で表示
- **ピラミッド図／空・雨・傘の3カラム図／イシューツリー図／提案4工程フロー** などを HTML/CSS で図解
- **良い例／悪い例の対比**、**コードレビューとの比喩表**、**要点コールアウト**で要点を明確化
- 各回に **演習（研修中）** と **宿題（実務適用週）**、全回で **AI併用** を明記
